## [1.0.0] - 2025-01-01

### Added

- OAuth authentication flow with automatic token refresh
- Portfolio management tools (holdings, positions, funds)
- Order management (place, modify, cancel orders)
- Market data tools (quotes, OHLCV, market status)
- Options analysis tools (chain, Greeks, strategy builder)
- GTT (Good Till Triggered) order support
- Brokerage and charge calculation tools
- Account information tools
- Comprehensive error handling with retry logic
- Full test suite with vitest
